about

UI-Utils is an updated FabricMC Minecraft modification that adds some features to inventory GUI(s) that can help with debugging plugins and glitch hunting. The mod is updated whenever a new update is released for Minecraft (including sub-releases but not including snapshots), and also whenever a new feature is added, although, this is more inconsistent and these features may be added after a new Minecraft update. There may be potential delay when updating this mod. Feel free to contribute to this project on Github using a pull request. This mod is not tested on any other OS other than Windows 10, it is unknown if Linux works, but, MacOS usually does not work, most likely due to issues with Java Swing.
